# Smoke Test: 影像查看器基础功能

**Tags:** smoke, viewer, cornerstone

## 前置条件

- 系统已启动 (http://localhost:5173)
- 已登录管理员账号
- 至少有一个检查记录

---

### Step 1: 进入检查列表

- **操作:** 导航到 `/studies`
- **验证:**
  - [ ] URL 包含 `/studies`
  - [ ] 页面显示检查列表
  - [ ] 列表有数据

### Step 2: 打开查看器

- **操作:** 点击第一个检查记录
- **验证:**
  - [ ] URL 变为 `/viewer/:id`
  - [ ] Cornerstone 视口加载
  - [ ] 工具栏显示

### Step 3: 测试工具切换

- **操作:** 按数字键 1-6 切换工具
- **验证:**
  - [ ] 按 1 切换到平移工具
  - [ ] 按 2 切换到缩放工具
  - [ ] 按 3 切换到窗宽窗位
  - [ ] 按 4 切换到长度测量
  - [ ] 按 5 切换到角度测量
  - [ ] 按 6 切换到探针

### Step 4: 测试缩放

- **操作:** 按 + 和 - 键缩放图像
- **验证:**
  - [ ] 图像缩放正常
  - [ ] 按 0 适配窗口

### Step 5: 测试帮助面板

- **操作:** 按 ? 键
- **验证:**
  - [ ] 快捷键帮助对话框显示
  - [ ] 包含所有快捷键说明
  - [ ] 关闭对话框

### Step 6: 测试多帧播放（如有）

- **操作:** 如果是多帧图像，测试 Cine 播放
- **验证:**
  - [ ] 播放按钮可用
  - [ ] 空格键切换播放/暂停
  - [ ] FPS 滑块可调整
  - [ ] 帧进度条可拖拽
