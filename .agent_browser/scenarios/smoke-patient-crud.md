# Smoke Test: 患者管理基础流程

**Tags:** smoke, patient, crud

## 前置条件

- 系统已启动 (http://localhost:5173)
- 已登录管理员账号

---

### Step 1: 进入患者列表

- **操作:** 导航到 `/patients`
- **验证:**
  - [ ] URL 包含 `/patients`
  - [ ] 页面显示患者列表
  - [ ] 列表有数据

### Step 2: 新增患者

- **操作:** 点击「新增患者」按钮
- **数据:**
  - MRN: `{{mrn}}`
  - 姓名: `{{name}}`
  - 性别: 男
  - 出生日期: `{{date}}`
  - 电话: `{{phone}}`
- **验证:**
  - [ ] 表单显示正确
  - [ ] MRN 输入框可编辑
  - [ ] 姓名输入框可编辑

### Step 3: 提交表单

- **操作:** 点击「保存」按钮
- **验证:**
  - [ ] 提交成功提示
  - [ ] URL 变为 `/patients` 或 `/patients/:id`
  - [ ] 新患者出现在列表中

### Step 4: 查看患者详情

- **操作:** 点击刚创建的患者
- **验证:**
  - [ ] 详情页显示患者姓名
  - [ ] MRN 显示正确
  - [ ] 电话显示正确

### Step 5: 编辑患者

- **操作:** 点击「编辑」按钮
- **数据:**
  - 电话: `1390000{{random}}`
- **验证:**
  - [ ] 表单显示当前数据
  - [ ] 修改后保存成功
  - [ ] 更新后的数据显示正确

### Step 6: 返回列表

- **操作:** 点击「返回」或导航到 `/patients`
- **验证:**
  - [ ] 列表显示更新后的数据
  - [ ] 患者数量正确
